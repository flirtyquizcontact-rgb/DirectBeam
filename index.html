Android to Android

Transfer selected files directly between compatible Android phones and tablets. Supported sessions use Secure Transport V6 with authenticated AES-256-GCM encryption and integrity verification.

Apple & Web Transfer

Transfer files between Android and iPhone, iPad, Mac or another browser-enabled device. Scan the QR code and open the temporary local session in Safari or another compatible browser.

PC Connect

Upload or download authorized files through a browser on the same trusted Wi-Fi network. No cable or desktop application is required.

Group Beam

Send selected files to multiple compatible nearby receivers and monitor progress separately for each connected device.

Resume and integrity

Supported interrupted transfers can resume without restarting every completed part. File integrity checks help detect incomplete or altered transfer results.

No DirectBeam cloud

Files selected for local transfer are exchanged between devices chosen by the user and are not uploaded to a cloud-storage server operated by DirectBeam.

Local browser transfer for Apple devices

DirectBeam can temporarily turn the Android device into a local transfer host. An iPhone, iPad, Mac, PC or other compatible device can connect by scanning the QR code or opening the displayed local address.

  • Use the same trusted Wi-Fi network or a DirectBeam local-only hotspot.
  • Open the transfer page in Safari or another compatible browser.
  • Select files on the sending device or download files authorized on Android.
  • No DirectBeam application is required on the browser device.
  • No internet connection is required when using the local-only transfer path.
Browser-transfer security: Apple & Web Transfer and PC Connect use a temporary authenticated local HTTP session. They do not use the same end-to-end encryption model as supported Android-to-Android Secure Transport V6 sessions. Use browser transfer only on a trusted private network and do not share the QR code, local address or session token with unauthorized persons.

Built around user control

You choose the files, folders and devices involved in every transfer. DirectBeam uses Android permissions and system file pickers so that access can be authorized only when required for a selected feature.

Important: DirectBeam does not require a DirectBeam account and does not operate a DirectBeam cloud-storage service for the files you select for local transfer.

Core functions

  • Send and receive files between compatible Android devices.
  • Wi-Fi Direct and compatible local-network transfer methods.
  • Apple & Web Transfer for iPhone, iPad, Mac and browsers.
  • Optional local-only hotspot for transfers without internet access.
  • Group Beam for supported multi-device transfers.
  • PC Connect for local browser-based file exchange.
  • Live progress, supported resume and integrity verification.
  • Transfer history and local diagnostics.
  • Optional DirectBeam Pro features through Google Play Billing.
