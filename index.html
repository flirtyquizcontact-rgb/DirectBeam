Android to Android

Direct device-to-device transfers with authenticated Secure Transport V6 and AES-256-GCM encryption.

Group Beam

Share selected files with multiple nearby Android receivers while keeping progress and retry status visible per device.

PC Connect

Transfer files through a browser on the same trusted local network using a temporary authenticated session.

Built around local transfer

DirectBeam does not require a DirectBeam account and does not upload your selected transfer files to a DirectBeam cloud server. Files are transferred between the devices you choose.

Important: PC Connect uses an authenticated local HTTP session and should be used only on a trusted private network. Android-to-Android Secure Transport V6 and PC Connect use different transport security models.

Core functions

  • Send and receive files directly between compatible Android devices.
  • Wi-Fi Direct as the primary one-to-one transport, with compatible local-network fallback.
  • Group Beam for supported multi-device transfers.
  • PC Connect for browser-based local transfer.
  • Transfer progress, integrity verification and supported resume workflows.
  • Optional DirectBeam Pro purchase through Google Play Billing.
